# Flutter 4-Day Beginner Workshop 🚀

This workshop is designed to introduce beginners to Flutter by building small apps step-by-step.  
Each day focuses on practical implementation with clear concepts and hands-on coding.

---

## 📅 Workshop Plan

### Day 1 – Event App (UI Basics)
Build a static event information app.
- Understanding Flutter project structure
- Widgets (Scaffold, Column, Row, Container, Text, Image)
- Layout and basic UI design

---

### Day 2 – Counter & To-Do App (State & Interaction)
Make apps interactive.
- Stateless vs Stateful widgets
- setState()
- Button interactions
- TextField & ListView
- Building a simple To-Do app

---

### Day 3 – Notes App (Local Storage)
Build a notes app with persistent storage.
- Working with lists
- Basic app structure
- Using SharedPreferences for local data storage

---

### Day 4 – Guided Mini Build
Team-based mini project.
- Apply UI, state, navigation, and storage
- Build a small app in groups
- Demo and discussion

---

Happy Coding 💙
